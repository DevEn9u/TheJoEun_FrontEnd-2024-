섹션제목

스페인의 아틀레티코 마드리드가 극적으로 유럽축구연맹(UEFA) 챔피언스리그(UCL) 8강에 올랐다. 14일(이하 한국시간) 아틀레티코 마드리드 홈구장인 스페인 마드리드 시비타스 메트로폴리타노에서 열린 2023-2024시즌 유럽 챔피언스리그 16강 2차전에서 이탈리아의 인터 밀란을 승부차기 끝에 이겼다.
